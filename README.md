## GESTIÓN DE HOTEL - USO POR TERMINAL
El sistema desarrollado consiste en una aplicación de gestión hotelera implementada en Python para administrar las operaciones básicas de un hotel mediante una interfaz de consola.

El programa permite gestionar habitaciones y huéspedes, incluyendo funcionalidades como consulta de habitaciones disponibles, registro de huéspedes mediante procesos de check-in, realización de check-out, visualización de huéspedes alojados y generación de estadísticas generales del hotel.
El sistema funciona con la generación de dos archivos de textos que se crean a la hora de ejecutarlo, con el fin de guardar la información de las habitaciones y los huéspedes en los mismos.

## INSTRUCCIONES DE USO
* Ejecutar 'hotel.py' en terminal (CMD, Powershell, etc)
* Se desplegará el menú del sistema, el usuario debe ingresar el número de la opción que quiera correr y darle enter
* Para salir del sistema, el usuario debe apretar el número 6 del teclado numérico y darle enter.
